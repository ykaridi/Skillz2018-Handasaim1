Bomber

Use case

This plugin is used for pirates carrying a bomb and for calculating wether to bomb an object.

Tip

This plugin should be stacked as the highest priority base plugin on every pirate
This way each pirate calculates if there is an object that is should bomb and acts according to it's directions to maximize damage of enemy pirates when it's carrying a bomb

Usage

  • Constructor #1
  • Constructor #2
Field Description
bombingScorer Scores each space object. any score <= 0 will be assumed not worthy bombing
selfBombingScorer Scores bombing the current pirate. a score <= 0 will be assumed not worthy bombing

The object with the maximal score will be bombed (assuming it's score is > 0)

Field Description

Will not bomb anything, only attempt maximizing the damage when carrying a bomb
