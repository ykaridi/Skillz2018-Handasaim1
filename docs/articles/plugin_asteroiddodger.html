AsteroidDodger

Use case

A plugin used for searching for nearby asteroids and acting according to the danger posed by them.
The plugin pushes the asteroid by a predefined pushing function, and halts the pirate if the asteroid is too far to push but poses imminent danger.

Tip

This plugin should be stacked as the highest priority base plugin on every pirate

Field Description
DangerRange Indicates the range (in turns) for an asteroid to be considered a danger, Any asteroid that will hit you within the specified range of turns will be considered a danger
HaltingRange Indicates the range (in turns) for an asteroid to be considered posing imminent threat and thus cause the pirate to stop until it can push the asteroid

Asteroids will be pushed to the opposite direction of the pirate.
