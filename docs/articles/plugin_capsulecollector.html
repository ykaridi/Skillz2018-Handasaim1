BoostedCapsuleSquadPlugin

Use case

A plugin used on attack groups to attempt scoring a certain capsule.
The group will split to two sub-groups:

  • Capsule scorers: these pirates will attempt bringing the capsule to the mothership (all non-carriers will attempt boosting the carrier)
  • Capsule campers: these pirates will attempt reaching the capsule and grabbing it (or waiting for it to respawn)

Usage

  • Constructor #1
Field Description
Capsule The capsule the group is in charge of
AvodingDistance The distance to attempt saving from enemy bunkers
MaxPushes An upper limit to the maximal number of simontanious pushes
CampDuration Number of turns an enemy pirate will be considered a camper
IgnoreDeeperCamps If the flag is false, the plugin will always check if it needs to save the current push for a deeper enemy bunker (deeper = closer to the mothership)
BackwardsMultiplier A multiplier for how far to look for enemy pirates chasing us from behind
