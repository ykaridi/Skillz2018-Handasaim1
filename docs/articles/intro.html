Writing a bot

In order to write a bot we must implement the DoTurn function in MyBot.cs.
Generally speaking, we should implement a 'Handler': SquadPirateHandler or IndividualPirateHandler
Example (Assuming a class BasicStrategy implementing IndividualPirateHandler exists):

public void DoTurn(PirateGame game)
{
    try
    {
        IndividualPirateHandler handler;
        /* Update Game */
        Engine.Store.NextTurn();
        Engine.Update(game);

        /* Strategy Change Check */
        handler = new BasicStrategy(); // <------------- Strategy selection

        /* Display logs */
        Engine.PrintStatusLog();

        /* Play Strategy Selection */
        Engine.DoTurn(handler);

        /* Display logs */
        Engine.PrintActionLog();

        Engine.Store.Flush();
    }
    catch (System.Exception e)
    {
        /* Error Handling */
        game.Debug("Error!");
        game.Debug(e.StackTrace);
    }
}
